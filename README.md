# 🛡️ Vault-256 | Military-Grade Encryption Suite

**Vault-256** is a high-security credential management suite leveraging **AES-256 (CBC Mode)** and **PBKDF2** key derivation. Engineered as part of a professional **BSc in CSE** portfolio, this tool demonstrates the practical application of symmetric cryptography and data integrity.

---

## 🔒 Security Architecture
- **Encryption Standard**: AES-256 (Advanced Encryption Standard).
- **Cipher Mode**: Cipher Block Chaining (CBC) with unique IVs.
- **Key Derivation**: PBKDF2 with 100,000 iterations for high-entropy protection.
- **Payload Integrity**: Implementation of Padding (PKCS7) and Base64 encoding for secure transmission.
- **Interface**: Obsidian-style secure dashboard with real-time feedback.

## 🛠️ Technology Stack
- **Cryptography:** Python / PyCryptodome
- **Backend:** Flask (WSGI Server)
- **Frontend:** HTML5, Modern CSS (Inter & JetBrains Mono), JavaScript (ES6+)
